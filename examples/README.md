# Usage Examples

이 디렉터리는 Device Plugin 기준 예제를 담고 있습니다.

기본 설치에서는 리소스 이름이 장비 종류에 따라 달라집니다.

- `rebellions.ai/ATOM`
- `rebellions.ai/REBEL`

통합 모드에서는 `rebellions.ai/npu`를 사용하면 됩니다.
